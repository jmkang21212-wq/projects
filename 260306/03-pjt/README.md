# SSAFY Bank 프로필 및 포트폴리오 UI 구현
- HTML5, CSS3, 그리고 Bootstrap 5 프레임워크를 활용하여 SSAFY Bank 고객의 개인 프로필과 투자 포트폴리오 관리 페이지를 구현한 실습입니다.

### 기술 스택
- HTML5 / CSS3
- Framework: Bootstrap 5.3.8

### 트러블 슈팅 (Troubleshooting)

1. 반응형 컨테이너 구조화: 기기 크기에 맞춰 화면이 자연스럽게 변하는 반응형 웹을 만들기 위해, 전체 레이아웃의 뼈대가 되는 container를 어떻게 구성하고 중첩해야 할지 감을 잡기 어려웠습니다.

2. 이미지 Fluid 처리: 프로필 사진 등의 이미지가 화면 크기에 맞춰 깨지지 않고 유연하게 줄어들거나 커지도록(Fluid) 제어하는 부분에서 레이아웃이 틀어지는 문제가 발생했습니다.

### 해결 방법:
두 문제 모두 Bootstrap 공식 문서를 적극적으로 활용하여 해결

1. Grid 시스템 파트와 컨테이너 문서를 정독하여 화면 크기별 동작 원리를 이해하고 코드를 수정했습니다.

2. 이미지 관련 공식 문서를 참고하여 img-fluid 등의 적절한 반응형 이미지 클래스를 적용함으로써 문제를 깔끔하게 해결할 수 있었습니다.

### 느낀점 및 회고
1. Bootstrap의 효율성 체감: 
- CSS로 직접 일일이 스타일링을 할 때보다 Bootstrap이 제공하는 다양한 컴포넌트와 유틸리티 클래스(d-flex, m-4, shadow-sm 등)를 활용하니, 훨씬 빠르고 통일감 있는 디자인을 완성할 수 있었습니다.

2. 공식 문서의 중요성: 
- 트러블 슈팅 과정을 통해, 막히는 부분이 생겼을 때 다른 곳을 찾기보다 가장 먼저 공식 문서를 확인하는 습관이 얼마나 중요한지 깊이 깨달았습니다.

3. 페이지 간의 연결 이해:
- a태그를 활용해 fin_profile.html과 fin_portfolio.html 두 개의 페이지를 직접 연결해 보면서, 실제 웹사이트가 어떻게 라우팅되고 동작하는지 전체적인 흐름을 파악할 수 있는 좋은 경험이었습니다.
